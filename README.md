### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/wdev007/coodesh-fullstack-challenge-2021-api
   ```
2. Change the settings files
   ```sh
   cp .env.example .env
   ```
3. Customize the time to run the import job in `config.yaml`
   ```yaml
   scheduling:
     cron: '*/30 * * * *'
   ```
